본 프로젝트는 세종대학교 신입생들을 위한 공개채팅방이며, 제공하는 기능은

1. 세종대학교의 위치를 지도로 표시
2. 생성된 학과 공개채팅방의 목록
3. 다른 사용자와 채팅
4. 현재 사용자의 정보 표시
5. 각 학과에 대한 설명과 수강할 수 있는 과목 목록
6. 이메일 / 비밀번호 기반의 로그인 기능
7. 이메일 / 비밀번호 기반의 회원가입 

이 정도가 있으며 현재 다른 기능들도 추가 예정에 있습니다.
